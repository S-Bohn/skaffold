YAML anchors

Anchors can be defined by having top-level keys starting with a dot, e.g. .common_stuff: &alias_name. You can then reuse the value using *alias_name.

Jsonnet Support

Skaffold supports the Jsonnet data templating language to write the configuration. Requires to explicitly set the configuration file with jsonnet filename extension: e.g.: 

skaffold -f skaffold.jsonnet build
